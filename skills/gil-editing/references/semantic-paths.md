# Semantic Paths

The machine-readable source is `schemas/semantic-paths.json`.

Important top-level fields:

- `4`: prefab definitions
- `5`: scene instances
- `6`: category index
- `8`: preview instances
- `9`: UI primitive store
- `10`: nodegraph pool
- `27`: decoration store
- `46`: UI primitive auxiliary store

Category mapping types:

- `100`: prefab id
- `200`: scene object id
- `400`: preview object id
